 (1) NPM x NPX 
    - O NPM baixa, instala e gerencia os arquivos/modulos em uma pasta chamada node_modules dentro da pasta principal do seu projeto
    - O NPX executa os pacotes de forma direta, ou seja, quando você digita npx comando, ele primeiro vai procurar o pacote dentro da pasta node_modules, caso ele não ache, ele baixa o pacote da internet, executa o comando e depois o descarta.
